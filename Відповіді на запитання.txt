1. Що таке Git?
Git- це система керування версіями.

2. В якому статусі можуть перебувати файли при роботі з Git?
tracked / untracked.

3. Для чого використовується  команда add?
add приймає шлях файлу або директорії.

4. Для чого використовується  команда status?
status використовується для перевірки статусу файлів.

5. Для чого використовується  команда commit? 
commit використовується для збереження версій файлу в даний момент часу.

6. Для чого використовується  команда init?
init використовується для ініціалізації репозиторіїв.

7. Що необхідно писати в коментарях комітів?
Потрібно записати всі зміни з минулих версій.

8. Як переглянути список комітів? 
команда git log.

9. Для чого використовується  команда checkout?
checkout використовується для навігації між гілками в репозиторії.
